# About

This is the personal website of Lucien Cartier-Tilet.

It is written with Nuxt and is entirely open-source. You can find the
code source [on Github](https://github.com/Phundrak/phundrak.com) and
[on Gitea](https://labs.phundrak.com/phundrak/phundrak.com).

This website uses only one functional cookie to remember your
preferences in terms of color scheme, which won’t be usable in order
to track you in any way.
